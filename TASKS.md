# Forge AI Development Roadmap

## Rules

Complete ONE milestone at a time.

Do not start the next milestone until:

- Everything builds
- Tests pass
- Documentation is updated
- The code is refactored if needed
- A git commit is created

Every milestone must end with:

- Summary
- Files changed
- Tests written
- Remaining work

Never skip milestones.
